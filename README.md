# a10y Helm Charts

Helm charts for [a10y (Autonomously)](https://autonomously-io.github.io) — autonomously understands and restores your network.

## Usage

```sh
helm repo add a10y https://autonomously-io.github.io/charts
helm install a10y a10y/autonomously
```

## Charts

| Chart | Description |
|---|---|
| [autonomously](./charts/autonomously) | Main a10y platform chart with all dependencies |
